# apps/api — Deprecated

**apps/api is NOT part of the live Shopify flow.**

The Shopify storefront does not call this backend. Storefront requests for cart decisions are handled by **revstack** via the **Shopify App Proxy**. revstack is the canonical backend for the Shopify integration.

This API was intended for future multi-platform support (e.g. non-Shopify storefronts). It is **deprecated and unused**. Do not modify or deploy this backend unless multi-platform support is reintroduced.
