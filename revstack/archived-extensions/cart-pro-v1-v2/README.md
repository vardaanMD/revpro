# DEPRECATED — Cart Pro V1/V2 theme extension

This extension is **deprecated** and **not used** by the app. The live storefront uses the Cart Pro V3 extension only (`extensions/cart-pro/` with block `cart_pro_embed_v3.liquid`).

This folder contains the legacy V1/V2 Liquid blocks and assets (cart-pro.js, cart-pro-v2.js, etc.) that loaded runtimes from `/apps/cart-pro/snapshot/v2` and `/apps/cart-pro/bootstrap`. Those routes have been removed; see `archived/routes-v1-v2/`.

Kept for reference only. Do not enable this extension in new installs.
